Argument Mapper

Argument Mapper Syntax Guide

Top-level fields

TITLE: Main title
SUBTITLE: Optional subtitle
THEME: scholarly | dramatic
LAYOUT: debate | cascade
PALETTE: any descriptive label
BACKGROUND: paper | dark
EMPHASIS: strength

Node format

NODE: A1
SIDE: PRO | CON | NEUTRAL
TYPE: CLAIM | EVIDENCE | COUNTERCLAIM | REBUTTAL | CONTEXT | QUESTION
LABEL: Short node title
TEXT: Full argument text
STRENGTH: 1-5

Link format

LINK:
FROM: A2
TO: A1
RELATION: supports | challenges | rebuts | extends | contextualizes

Mini example

TITLE: Sample Map
LAYOUT: debate

NODE: P1
SIDE: PRO
TYPE: CLAIM
LABEL: Main point
TEXT: This is the central pro claim.
STRENGTH: 5

NODE: C1
SIDE: CON
TYPE: COUNTERCLAIM
LABEL: Objection
TEXT: This is a counterargument.
STRENGTH: 4

LINK:
FROM: C1
TO: P1
RELATION: challenges
